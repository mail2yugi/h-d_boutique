# Contributing to H & D Boutique

Thank you for your interest in contributing! This project is built with 100% free services.

## Development Setup

1. Clone the repository
2. Follow `SETUP.md` for complete setup instructions
3. Create a feature branch
4. Make your changes
5. Submit a pull request

## Code Style

- TypeScript for type safety
- ESLint + Prettier for formatting
- Follow existing patterns

## Testing

- Test all features locally before submitting
- Ensure no paid services are introduced
- Verify mobile app works via Expo Go

## Pull Request Process

1. Update README.md if needed
2. Ensure all apps build successfully
3. Test on both web and mobile
4. Describe changes clearly in PR
